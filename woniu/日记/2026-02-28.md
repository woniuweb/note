#算法 
## 删除中间节点

1 -> 2 -> 3 -> 4 -> 5 -> 6

只给中间节点，获取不到头节点，比如 拿到4中间节点，从后向前覆盖，最后断开最后一个节点。

```
class Solution {
    public void deleteNode(ListNode node) {
        while(node.next.next != null) {
            node.val = node.next.val;
            node = node.next;
        }
        node.val = node.next.val;
        node.next = null;
    }
}
```

2. 正确做法，**链表可以 改指针**，不需要迁移。

```
 public void deleteNode(ListNode node) {
        node.val = node.next.val;
        node.next = node.next.next;
}
```


## 分割链表

给你一个链表的头节点 head 和一个特定值 x ，请你对链表进行分隔，使得所有 小于 x 的节点都出现在 大于或等于 x 的节点之前。 

思路：直接操作链表，双指针同时构建x左右两边的链表，最后相连，这代表需要创建新链表来保存两个，创建两个新空链表来操作。
注意：
1. 并且需要防止 large .next为空，防止链表闭环。
2. 最后输出和相接时，需要.next,去除创建的头空节点。


```
class Solution {
    public ListNode partition(ListNode head, int x) {
        ListNode head1 = new ListNode(0);
        ListNode head2 = new ListNode(0);

        ListNode small =head1;
        ListNode large  =head2;

        while(head != null){
            if(head.val < x){
                small.next = head;
                small = small.next;
            } else {
                large.next = head;
                large = large.next;
            }
            head = head.next;
        }
        //防止链表循环
        large.next = null;
        // 两队相接
        small.next = head2.next;

        return  head1.next;

    }
}
```


## 删除链表的倒数第 N 个节点
给你一个单链表的头节点 head，请你删除链表的倒数第 n 个节点，并返回链表的头节点。

**0 ->** 1 -> 2 -> 3 -> 4 -> 5 -> 6
再增加头节点，防止删除head节点时报错，本质通过双指针间隔n,来找倒数第n+1个节点。

```
class Solution {
    public ListNode deleteNode(ListNode head, int x) {
        ListNode dummy = new ListNode(0);
        dummy.next = head;

        ListNode dummy1 = dummy;
        ListNode dummy2 = dummy;

        for (int i = 0; i < x; i++) {
            dummy2=dummy2.next;
        }
        while(dummy2.next != null ){
            dummy1 = dummy1.next;
            dummy2 = dummy2.next;
        }
        dummy1.next = dummy1.next.next;
        return dummy.next;
    }
}
```



## 链表求和
给定两个用链表表示的整数，每个节点包含一个数位。  这些数位是反向存放的，也就是个位排在链表首部。  编写函数对这两个整数求和，并用链表形式返回结果。 

输入：(7 -> 1 -> 6) + (5 -> 9 -> 2)，即617 + 295
输出：2 -> 1 -> 9，即912

思路：依旧创建头节点dummy用于返回最后结果，再创建移动操作节点tail，用于指针设置和赋值。
需要考虑进位，两个链表是否为空的情况，不为空继续next计算，结果赋值给tail.val ,继续tail = tail.next;


```
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode dummy = new ListNode(0);
        ListNode tail = dummy;
        int carry = 0;
        while (l1 != null || l2 != null ||carry != 0){
            int num =0;
            if (l1 != null) {
                num += l1.val;
                l1 = l1.next;
            }

            if (l2 != null) {
                num += l2.val;
                l2 = l2.next;
            }

            num += carry;

            carry = num / 10;
            tail.next =  new ListNode(num % 10);
            tail = tail.next;
        }
        return dummy.next;
    }
}
```


## 链表求和2
第二关：正向存放怎么办？

输入：  l1 = 7 -> 2 -> 4 -> 3  ，l2 = 5 -> 6 -> 4
输出：7 -> 8 -> 0 -> 7

思路：用栈，将链表压栈，pop出计算，最后使用**头插法**，进行链表构建。
注意：链表只是有next来指向下一个节点，节点之间可以互相指向，不分前后。


```
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode head = null;
        Deque<Integer> stack1 = new ArrayDeque<Integer>();
        Deque<Integer> stack2 = new ArrayDeque<Integer>();
        while(l1 != null){
            stack1.push(l1.val);
            l1 = l1.next;
        }
        while(l2 != null){
            stack2.push(l2.val);
            l2 = l2.next;
        }
        int carry = 0;
        while(!stack1.isEmpty() || !stack2.isEmpty() || carry>0){
            int num =0;
            if(!stack1.isEmpty()){
                Integer pop = stack1.pop();
                num +=pop;
            }
            if(!stack2.isEmpty()){
                Integer pop = stack2.pop();
                num +=pop;
            }
            num += carry;
            carry = num/10;
			
			// 头插法，关键
            ListNode node = new ListNode(num%10);
            node.next = head;
            head = node;
        }
        return head;
    }
}
```
