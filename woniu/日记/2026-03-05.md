## 日记
### 情绪泛化（Emotional Flooding）
当情绪上来时，大脑会自动做三件事：
1️⃣ 只记得对方做错的事
2️⃣ 忽略对方做过的好事
3️⃣ 用极端词汇
所以会出现：
“你什么都没改”，“你从来不做”，“你什么都没做过”
这些话90%不是事实判断，而是情绪表达。
### 事情经过（ai阐述版）
今天早上因为早餐和做事节奏的问题发生了一次争执。当时我先起床下楼，收拾了桌子、倒了垃圾，也整理了一下厨房。媳妇在楼上化妆准备出门。因为我做事节奏比较慢，她下来后说我“太慢了，看着难受”，情绪有些急躁。
随后我们围绕早饭的问题产生分歧。我的想法是早上时间紧张，谁先起谁简单烧水或泡面即可，不一定需要专门做早餐；但她似乎更希望我能主动帮忙准备一些。
在沟通中，我表达了自己平时晚上也会收拾屋子、做一些家务，希望她不要因为早上看到我动作慢就对我发脾气。我也提到我做家务时并没有要求她一起做。
谈到这里她情绪上来了并哭了，说我“什么都没改过”。我当时感到比较无助，因为自己觉得平时确实在承担一些家务，但在她情绪上来时似乎没有被看到。
### 分析
首先,我:任务型统计  ，她:情绪性统计。当他情绪表达，而我罗列我的行为，逻辑方式无法响应对方情绪，对方在表达情绪，沟通存在错位。
早上问题分析：
1. 她：“你什么都没做”。“你太慢了”，她想表达的是：**我希望被照顾一点，我早上很忙很焦虑**，他不会直接说，所以你听到的是 指责，但她其实在表达 **需求**。
2. 早上她赶时间，心里焦躁，如果你没有帮早餐，她就会感觉：**我一个人在忙**。于是之前做的事情被情绪覆盖了。
3. 付出统计方式不同：我:任务型统计  ，她:情绪性统计。她关注的是：**在我难受的时候你有没有帮我**。

性格：对时间敏感，希望被照顾。
”早餐其实是一个 情感象征“，需要多的情绪照顾，不光琐事，还需时不时准备**惊喜**。
### 解决方向
-沟通先针对情绪，后是逻辑
-时间紧迫时，不要刺激对方
-设计规则，安排每天分工？

<u>不是要每天都做什么，而是要对方感到被体贴照顾。</u>


## 算法

### 动物收容所
动物收容所（队列设计）收容所只收 猫(0) 和 狗(1)，遵循 先进先出 FIFO。
操作：
enqueue([id,type])：加入动物
dequeueAny()：领养最早进入的动物
dequeueDog()：领养最早进入的狗
dequeueCat()：领养最早进入的猫
返回值：[id,type]，若不存在返回 [-1,-1]。常见实现：两个队列 dogQueue、catQueue，比较队头决定 dequeueAny。允许使用Java内置的LinkedList数据结构。 

#### 编号比较
第一思路：维护两个队列，一猫一狗，但是我第一次知道linkedList竟然有栈和队列的方法，不愧是链表。

```
class AnimalShelf {
    LinkedList<int[]> dogQueue;
    LinkedList<int[]> catQueue;
    public AnimalShelf() {
        dogQueue = new LinkedList<int[]>();
        catQueue = new LinkedList<int[]>();
    }
    
    public void enqueue(int[] animal) {
        int type = animal[1];
        if (type == 0) {
            catQueue.add(animal);
        }else {
            dogQueue.add(animal);
        }
    }

    public int[] dequeueAny() {

        if (dogQueue.isEmpty() && catQueue.isEmpty()) {
            return new int[]{-1, -1};
        }

        if (dogQueue.isEmpty()) {
            return catQueue.poll();
        }

        if (catQueue.isEmpty()) {
            return dogQueue.poll();
        }

        int dogId = dogQueue.getFirst()[0];
        int catId = catQueue.getFirst()[0];

        if (dogId < catId) {
            return dogQueue.poll();
        } else {
            return catQueue.poll();
        }
    }
    
    public int[] dequeueDog() {
        if (dogQueue.isEmpty()) {
            return new int[]{-1, -1};
        }
        int[] first = dogQueue.getFirst();
        dogQueue.remove(first);
        return first;
    }
    
    public int[] dequeueCat() {
        if (catQueue.isEmpty()) {
            return new int[]{-1, -1};
        }
        int[] first = catQueue.getFirst();
        catQueue.remove(first);
        return first;
    }
}
```

#### 时间戳解法

```
import java.util.LinkedList;

class AnimalShelf {

    LinkedList<int[]> dogQueue = new LinkedList<>();
    LinkedList<int[]> catQueue = new LinkedList<>();
    int order = 0;

    public AnimalShelf() {

    }

    public void enqueue(int[] animal) {
        int[] node = new int[]{animal[0], animal[1], order++};

        if (animal[1] == 0) {
            catQueue.add(node);
        } else {
            dogQueue.add(node);
        }
    }

    public int[] dequeueAny() {

        if (dogQueue.isEmpty() && catQueue.isEmpty()) {
            return new int[]{-1,-1};
        }

        if (dogQueue.isEmpty()) {
            int[] cat = catQueue.poll();
            return new int[]{cat[0], cat[1]};
        }

        if (catQueue.isEmpty()) {
            int[] dog = dogQueue.poll();
            return new int[]{dog[0], dog[1]};
        }

        int[] dog = dogQueue.peek();
        int[] cat = catQueue.peek();

        if (dog[2] < cat[2]) {
            dog = dogQueue.poll();
            return new int[]{dog[0], dog[1]};
        } else {
            cat = catQueue.poll();
            return new int[]{cat[0], cat[1]};
        }
    }

    public int[] dequeueDog() {
        if (dogQueue.isEmpty()) {
            return new int[]{-1,-1};
        }
        int[] dog = dogQueue.poll();
        return new int[]{dog[0], dog[1]};
    }

    public int[] dequeueCat() {
        if (catQueue.isEmpty()) {
            return new int[]{-1,-1};
        }
        int[] cat = catQueue.poll();
        return new int[]{cat[0], cat[1]};
    }
}
```
