## 算法
### 三步问题
//三步问题。有个小孩正在上楼梯，楼梯有 n 阶台阶，小孩一次可以上 1 阶、2 阶或 3 阶。实现一种方法，计算小孩有多少种上楼梯的方式。结果可能很大，你需要
//对结果模 1000000007。 

思路：斐波那契数列，但是递归会超时   时间 O(n)   空间 O(n)。需要滚动 DP。

f(n)=f(n-1)+f(n-2)+f(n-3)

```
class Solution {

    int MOD = 1000000007;
    int[] memo;

    public int waysToStep(int n) {
        memo = new int[n + 1];
        return dp(n);
    }

    public int dp(int n) {
        if(n <=2){
            return n;
        }else if(n == 3){
            return 4;
        }
        memo[n] = (dp(n-1) + dp(n-2) + dp(n-3)) % MOD;
        return memo[n] ;
    }
}
```

## 迷路的机器人
一个机器人位于r × c网格的左上角（坐标 [0, 0]），需要移动到右下角（坐标 [r-1, c-1]）。机器人每次只能向下或向右移动一步，且不能进入有障碍的格子。
输入格式：
grid: 一个 r × c 的二维整数数组，0 表示空格子（可通行），1 表示障碍物（不可通行）
输出格式：
返回一条从左上角到右下角的可行路径，路径表示为坐标数组 [[r₁,c₁], [r₂,c₂], ..., [rₙ,cₙ]]，其中每个元素是 [行号, 列号]，如果没有可行路径，返回空数组 []
约束条件：
1 ≤ r, c ≤ 100，grid[0][0] 和 grid[r-1][c-1] 保证为 0（起点和终点无障碍），路径必须按移动顺序排列，起点必须是 [0, 0]，终点必须是 [r-1, c-1]

思路：这种记忆dps，主要是记忆走过的节点+


```
class Solution {
    List<List<Integer>> path = new ArrayList<>();
    boolean[][] visited;

    public List<List<Integer>> pathWithObstacles(int[][] grid) {
        int r = grid.length;
        int c = grid[0].length;

        visited = new boolean[r][c];

        if(dfs(grid,0,0)){
            return path;
        }

        return new ArrayList<>();
    }

    private boolean dfs(int[][] grid,int i,int j){
        int r = grid.length;
        int c = grid[0].length;

        if(i>=r || j>=c){
            return false;
        }
        if(grid[i][j] == 1){
            return false;
        }
        if (visited[i][j] == true) {
            return false;
        }
        // 记录走过的节点
        visited[i][j] = true;
        path.add(Arrays.asList(i, j));

        //终点
        if(i == r - 1 && j == c - 1){
            return true;
        }
        //右
        if(dfs(grid, i, j+1)){
            return true;
        }
        if(dfs(grid, i+1, j)){
            return true;
        }

        //右，下都不通，回溯
        path.remove(path.size() - 1);
        return false;
    }
}
```
