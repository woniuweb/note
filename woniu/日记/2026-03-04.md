## 栈排序

要求：用两个栈实现有序栈，最小元素在栈顶

| 要点     | 说明                    |
| :----- | :-------------------- |
| 限制     | 只能用1个辅助栈，不能用数组等其他结构   |
| 目标     | 栈顶始终是最小元素             |
| 操作     | push、pop、peek、isEmpty |
| 空栈peek | 返回 -1                 |

思路：一个做主栈，另一个做辅助，push的时候，做为临时放置元素的存在，主栈一直保持 从小到大的顺序。
```
class SortedStack {
    Deque<Integer> stack;
    Deque<Integer> temp;
    public SortedStack() {
        stack = new ArrayDeque<>();
        temp = new ArrayDeque<>();
    }
    
    public void push(int val) {
        while(!stack.isEmpty() && stack.peek() < val) {
            temp.push(stack.pop());
        }
        stack.push(val);
        while(!temp.isEmpty()){
            stack.push(temp.pop());
        }

    }
    public void pop() {
        if (!stack.isEmpty()) {
            stack.pop();
        }
    }
    
    public int peek() {
        return stack.isEmpty() ? -1 : stack.peek();
    }
    
    public boolean isEmpty() {
        if(stack.isEmpty()){
            return true;
        }
        return false;
    }
}
```
